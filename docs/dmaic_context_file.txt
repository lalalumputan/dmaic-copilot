📌 1. Project Overview

DMAIC Copilot adalah aplikasi agentic AI berbasis Python + LLM + Streamlit yang berfungsi sebagai digital Continuous Improvement engineer (Lean Sis Siga Master Black Belt) untuk menjalankan proyek Lean Six Sigma DMAIC (Define, Measure, Analyze, Improve, Control).

Sistem ini memiliki:

Modular multi-phase agents (Define → Control)

LLM reasoning for interpretation & decision-making

Analytics engine for OEE, defect rate, Pareto, CTQ, DOE, RCA

Persistent memory per project

Audit logging

Phase reports + full DMAIC report generator

(Upcoming) Agent Memory Layer & Feedback-based Adaptation

2. Goals of the System
Primary Goals

Membantu user mengeksekusi proyek DMAIC secara sistematis dan terstruktur

Menghasilkan insight berbasis data dari file CSV maupun text input

Memberikan rekomendasi berbasis LLM untuk setiap fase.

Menghasilkan laporan standar DMAIC secara otomatis.

Mengelola multi-project dengan state persistence.

Menyiapkan arsitektur untuk agentic AI adaptation.

🎯 Secondary Goals

Mudah di-maintain dan scalable.

Modular, sehingga setiap fase dapat dikembangkan terpisah.

Dapat di-upgrade menjadi true agentic AI dengan memory & feedback loop.

3. System Architecture Summary
🔹 A. Data Processing Layer

Melakukan perhitungan dan analitik DMAIC:

CTQ extraction

VOC clustering (opsional)

Defect rate summary

OEE

Stratifikasi

Pareto analysis

RCA pattern extraction

DOE planner (factor-level matrix generation)

🔹 B. AI Agent Layer

Setiap fase memiliki agent:

Phase	Agent Name	Function
Define	define_agent()	Charter, SIPOC, CTQ, problem statement refinement
Measure	measure_msa_oee()	OEE, defect summary, measurement analysis
Analyze	analyze_rootcause()	Pareto insight, RCA, hypothesis generation
Improve	improve_doe() / improve_kaizen()	DOE matrix or Kaizen plan
Control	control_plan()	Control plan, monitoring strategy

Semua agent:

menerima input data dan context

menghasilkan structured output + AI Insight

disimpan ke project memory

📌 9. Prompt Style Guidelines (important!)
All prompts must:

Use DMAIC logic

Use lean/practical manufacturing language

Provide actionable insight

Avoid generic coaching language

Use structured output (tables, bullet, steps)

Each phase must include:

Define → Charter, CTQ, SIPOC
Measure → OEE, MSA, defect analysis
Analyze → Hypothesis, RCA, prioritization
Improve → DOE/Kaizen plan + expected impact
Control → Control plan + monitoring KPIs

📌 11. Glossary (for ChatGPT parsing)

Agent = function/module that runs one DMAIC phase with LLM reasoning.

Project memory = folder containing JSON per phase.

Episode = one execution of an agent (future adaptation).

AI Insight = LLM interpretation of analytics.

Audit log = chronological list of agent actions.
